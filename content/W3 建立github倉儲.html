W3 建立github倉儲

到老師的 cad2021_hw 倉儲 template 建立一個自己的 cad2021_hw 倉儲 → 建立完後開啟 ipv4 啟動檔 clone 進 tmp 裡面

git clone --recurse-submodules 倉儲網址.git

 → 到 github 的設定裡

找到 Developer settings 

點進去找到 tokens 並創建一個 tokens 

串建完成後複製他給的 personal access 

到 tmp → 自己的倉儲資料夾 → .git → config 拉進 SciTE

貼上剛剛複製的那段後面加一個@，@後面是自己的倉儲網址
